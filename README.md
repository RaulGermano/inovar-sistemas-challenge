## Getting Started

Instructions:

```bash
#to install dependencies
npm install

#to build in local
npm run dev

```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build tests

After runs the application on dev environment `npm run dev` execute:

```bash
npm run test
```
